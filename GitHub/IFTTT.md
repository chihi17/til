# IFTTT * GitHub
IFTTTとGitHubを連携した際に使える"Add ingredient"の説明

## Premise

[IFTTT](https://ifttt.com/p/chiro17t)

## Result

>{{IssueTitle}}  

Issueのタイトル  

>{{CreatedAt}}  

Issueの作成時間  
ex: December 11, 2018 at 02:00PM  

>{{IssueURL}}  

IssueのURL  
ex: ift.tt/xxxxx  

>{{IssueLabels}}  

ラベルつけずにテストしたため表示形式が不明  

>{{RepositoryName}}  

レポジトリ名  
GitHubPagesのレポジトリの場合、リンクされる  

>{{AssigneeUsername}}  

アサイン指定されていないとき、何も表示されない  

>{{AuthorUsername}}  

IDのみ登録の状態のとき、IDが表示される  
登録名については不明  

>{{AuthorAvatarImageURL}}  

プロフィール写真  
ex: ift.tt/xxxxx  

>{{IssueClosedAt}}  

IssueのClose時間  

>{{IssueBody}}  

Issue本文
